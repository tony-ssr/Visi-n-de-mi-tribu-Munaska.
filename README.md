## 🌟 Descripción

*Visión Munaska* es una **Single Page Application (SPA) estática** construida con HTML5, CSS3 y Vanilla JavaScript, conectada en tiempo real a un backend de **Supabase**. Diseñada con temática de abejas 🍯 y paleta de colores dorada, la aplicación permite visualizar y gestionar el progreso hacia la meta de **93 vidas enroladas**.

---

## ✨ Funcionalidades

| Pestaña | Función |
|---|---|
| 🍯 **Visión Principal** | Contador animado, barra de progreso, panal hexagonal con 93 celdas |
| 📋 **Consultas** | Tarjetas con búsqueda libre y filtros por tribu y estado |
| ⚙️ **Gestión** | Formulario CRUD completo: Crear, Editar (modal) y Eliminar (con confirmación) |

## ⚙️ Configuración (IMPORTANTE)

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/tony-ssr/Visi-n-de-mi-tribu-Munaska..git
   cd Visi-n-de-mi-tribu-Munaska.
   ```

2. **Rellena tus credenciales de Supabase** en `assets/js/supabase.js`:
   ```js
   const SUPABASE_URL = 'https://tuproyecto.supabase.co';
   const SUPABASE_ANON_KEY = 'tu-anon-key-aqui';
   ```

3. **Abre `index.html`** directamente en tu navegador, o usa una extensión de servidor local como **Live Server** (VSCode).

```sql
CREATE TABLE vision_munaska (

    miembro_dca  TEXT NOT NULL,
    tribu_destino INTEGER NOT NULL,

    estado       TEXT NOT NULL
```
