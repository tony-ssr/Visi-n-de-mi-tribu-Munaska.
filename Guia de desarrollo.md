Actúa como un Desarrollador Frontend Experto. Necesito crear una aplicación web estática (HTML5, CSS3, Vanilla JavaScript) para gestionar la visión de enrolamiento de un programa de liderazgo. El proyecto se conectará a un backend de Supabase utilizando su SDK de JavaScript y estará optimizado para desplegarse en GitHub Pages.

**1. Temática y Diseño (UI/UX):**
*   **Tema:** Abejas ("Abejitas"). La tribu se identifica con este animal.
*   **Paleta de colores:** Tonos cálidos (amarillo miel, dorado, negro, blanco, gris claro).
*   **Estilo:** Moderno, limpio, con bordes redondeados (glassmorphism opcional), uso de sombras suaves. Debe ser completamente Responsive (Mobile-first).
*   **Iconografía/Imágenes:** Usa emojis de abejas 🐝 o SVGs minimalistas de panales/abejas para decorar las tarjetas y el header.

**2. Estructura de Carpetas Exigida:**
El proyecto debe estar modularizado con la siguiente jerarquía:
/
├── index.html
├── assets/
│   ├── css/
│   │   ├── style.css
│   │   └── components.css
│   ├── js/
│   │   ├── app.js (lógica principal y manejo de pestañas)
│   │   ├── supabase.js (configuración e inicialización del cliente)
│   │   └── api.js (funciones CRUD para interactuar con la tabla)
│   ├── img/ (para logos e ilustraciones de abejitas)
│   └── media/ (audios o videos futuros)

**3. Base de Datos (Supabase):**
La tabla ya existe y tiene la siguiente estructura:
CREATE TABLE vision_munaska (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    miembro_dca TEXT NOT NULL,
    tribu_destino INTEGER NOT NULL,
    nombre_enrolado TEXT NOT NULL,
    estado TEXT NOT NULL
);
*Nota para la IA: Utiliza CDN para importar el cliente de Supabase en el HTML: `<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>`*

**4. Arquitectura de Pestañas (Tabs):**
La interfaz debe ser de una sola página (SPA) manejada con JavaScript, dividida en 3 pestañas principales:

*   **Pestaña 1: Visión Principal (Dashboard)**
    *   Un gran contador central mostrando el número total de vidas enroladas frente a la meta (Ej: "54 / 93 Vidas").
    *   Una barra de progreso visual (estilo panal de abejas o llenado de miel) que calcule el porcentaje en base a 93.
    *   Diseño muy visual y motivador.

*   **Pestaña 2: Consultas (Directorio)**
    *   Mostrar los datos de la base de datos en formato de "Tarjetas" (Cards).
    *   Debe incluir un input de búsqueda de texto libre (para filtrar por `nombre_enrolado` o `miembro_dca`).
    *   Debe incluir `selects` para filtrar por `tribu_destino` (ej. 123, 124, 125, 126) y por `estado` (ej. 'Ya suma a la Visión', 'Pendiente inicio').

*   **Pestaña 3: Gestión (CRUD)**
    *   Un formulario limpio para **Crear** nuevos registros (Miembro DCA, Tribu Destino, Nombre Enrolado, Estado).
    *   Una tabla administrativa que liste todos los registros con dos botones de acción por fila: **Editar** (abre un modal para actualizar) y **Eliminar** (con alerta de confirmación).
    *   Validación de formularios en el frontend antes de enviar a Supabase.

**5. Instrucciones Finales para la IA:**
*   Genera el código completo para todos los archivos mencionados en la estructura.
*   En `supabase.js`, deja constantes vacías para `SUPABASE_URL` y `SUPABASE_ANON_KEY` para que yo las rellene localmente.
*   Asegúrate de que el cambio entre pestañas sea fluido y sin recargar la página.
